# Code of conduct

EdgeWatch contributors are expected to communicate professionally and respectfully.

## Expected behavior

- Discuss ideas and code without personal attacks.
- Give specific, actionable feedback.
- Respect privacy and never publish another person's credentials, network details, logs, or personal information.
- Report security issues privately through the process in SECURITY_POLICY.md.
- Accept corrections when evidence or testing shows that an assumption was wrong.

## Unacceptable behavior

Harassment, threats, discriminatory language, deliberate disclosure of private information, and knowingly submitting malicious code are not permitted.

## Enforcement

The project owner may edit, hide, reject, or remove contributions and may restrict participation when this policy is violated.
